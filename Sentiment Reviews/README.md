# Анализ тональности отзывов
Исследование проведено в рамках производственой практики ВУЗа.  

**Цель**: применение нейросетевых методов машинного обучения при работе с текстом.  
**Основная задача**: классификация отзывов по 3 классам.  

**Инструменты**: pandas, re, sklearn, keras, LSTM

Точность на тестовых данных: 72,5%


## Данные
Исходный набор данных для исследования: https://www.kaggle.com/datasets/mar1mba/russian-sentiment-dataset/code . Набор данных распространяется по лицензии MIT (Copyright (c) 2013 Mark Otto, Copyright (c) 2017 Andrew Fong).  

Набор данных содержит 290390 экземпляров текстов отзывов на русском языке с указанием источника, размеченных по трем классам тональности : 0 – нейтральный, 1 – положительный, 2 – отрицательный.  
Данные программно выгружаются через kagglehub.
